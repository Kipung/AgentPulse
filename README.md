# AgentPulse ⚡️

**Orchestrate. Monitor. Maximize.**

AgentPulse is a developer-first platform designed to help you **max out your usage of AI coding agents**.

I am building this because I use agents every day. The friction of managing them—losing context between terminals, missing completions, and flying blind on token usage—slows me down. I want a tool that makes using agents seamless, effective, and actually fun.

## Core Modules

AgentPulse is built on modular, single-purpose tools that solve specific friction points:

### 1. AgentNotifier (Focus) 🔔
* **"Am I done yet?"**
* **Problem:** You run a long task, switch tabs, and forget about it.
* **Solution:** Smart notifications that detect when an agent task completes or when you've lost focus, letting you reclaim your attention immediately.

### 2. AgentVitals (Sustainability) ❤️
* **"Can I keep going?"**
* **Problem:** You hit context limits unexpectedly or lose track of API costs across different sessions.
* **Solution:** Real-time tracking of:
    * **Token Usage:** How much you've spent.
    * **Context Window:** How much "memory" is left before the agent starts forgetting.
    * **Time:** Session duration and efficiency stats.

## The Mission
* **Unified Oversight:** One place to see what all your agents (Claude, Gemini, Codex) are doing.
* **Context Retention:** Never lose your train of thought when switching tools.
* **Zero Friction:** Wraps your existing CLI workflow; doesn't replace it.
